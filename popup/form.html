Para usar, entre no SIA, acesse Período Letivo > Início do Período > Planejamento das Disciplinas, selecione a disciplina e clique em alterar (ícone do lápis) e cole seu planejamento na caixa abaixo usando colunas delimitadas por caracteres como: espaço, ";", "|", ",", etc. Para funcionar, basta ser consistente. No exemplo a seguir é usado o ;: 

1;10/03/2017;Atividades para este horário
1;10/03/2017;Atividades para este horário
1;17/03/2017;Atividades para este horário
Os dados são, respectivamente, o período (bim ou sem com um dígito), data (no formato dd/mm/aaaa) e descrição da atividade.

Para concluir, clique em Preencher planejamento e veja a mágica acontecer ;)



Funcionalidades futuras: cálculo automático das datas, assistente para preenchimento das notas e faltas.

Tu podes fazer o planejamento no Excel, Calc ou Google spreadsheets e então pedir para exportar/salvar/baixar como CSV!
